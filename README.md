# COKE-SALES-ANALYSIS

## Project Overview
Analyzed Coke sales data using Excel to identify sales trends, regional performance, retailer performance, and beverage brand insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Dashboarding

## Key Analysis
- Total Sales by Retailer and Region
- Monthly Sales Trends
- Beverage Brand Performance
- Units Sold Analysis

## Insights
- Identified top-performing regions and retailers
- Compared beverage brand sales performance
- Analyzed monthly sales growth trends
